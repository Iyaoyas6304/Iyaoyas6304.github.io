IT Project (Creating Active Directory)

Step 1: In Virtual Box, create a "New" VM.

Name it "DC" for Domain Controller. In the setup of DC, change Version to "Other Windows (64-bit)" and click continue. For the Memory Size, it does depend how much RAM you have. Hopefully you 8Gb, but make it 2048Mb and click Next. On the next page, leave everything as is. Everything is set to "Defaults". Click Continue until DC has been created.

Step 2: DC Setup

Double click on your DC to start it. This is where we are going to install Microsoft Server 2019. MAKE SURE THAT YOUR COMPUTER IS SET TO HANDLE VIRTUALIZATION THROUGH THE BIOS OF YOUR COMPUTER. (VARIOUS MAKES OF COMPUTERS VARIES WHICH BUTTON TO HIT ON BOOTUP OF YOUR COMPUTER, BUT IT CAN BE EITHER: ESCAPE, F1,F2,F6,F8,F10, OR F12. WHEN INSIDE OF THE BIOS, GO TO "system configuration" AND "enable" "virtualization technology".

Step 3: 1st Time Start of DC

After the server OS has been installed, it will ask you to create a password for the Built-In Admin. Make the password "P@ssw0rd1". And click "Finish" to finalize the settings. When it asks you to "Press Ctrl+Alt+Delete", go to the input menu at the top, keyboard, and click "Insert Ctrl+Alt+Delete". Then put in the password that you've set, "P@ssw0rd1".

Step 4: Setting up the Networking

DC NIC1 (INTERNET) = (DHCP) WILL GET ADDRESSING FROM HOME ROUTER. DC NIC2 (INTERNAL) (THE VMWARE NETWORK): IP: 172.16.0.1, MASK: 255.255.255.0, GATEWAY: <empty>, DNS: 127.0.0.1 (this is a loop back address).

Step 5: Setting up the Domain

In the "Server Manager Dashboard", click on "Add roles and features. Click "Next" 3 times until you get to a list of checkboxes. Check the box that says "Active Directory Domain Services". Then click next until you get to the install option, and install. Once it's done installing, it will say "Configuration required. Installation succeeded on DC" underneath the progress bar, click close.

Step 6: Setting up the RAS/NAT

With you being logged in as the local admin, click on "Add roles and features", click "Next" until you see Server Roles. Check the "Remote Access" box, click next until you get to "Role Services". Click the "Routing" box, click "Add Features" on the next screen, click Next until you get to the "Install" button and click install. When it's done installing, you will see "Configuration required. Installation succeeded on DC" underneath the progress bar, click close.

Step 7: Setting up DHCP

Click on "Add roles and features", click "Next" until you get to the "Server Roles". Select "DHCP Server", accept all of the defaults by clicking "Next" until you see the button to "Install". Click the "Install" button. You know it's done when you see the message, "Configuration required. Installation succeeded on DC" underneath the progress bar, click close.
