    • Subnetting SOHO Networking Project with Packet Tracer

    Configured a SOHO network using Cisco devices to create three Class C subnets with VLANs for improved network segmentation and security.

    Objectives:

    • Design and implement a network topology with three distinct Class C subnets.
    • Configure routers and switches using Cisco IOS commands.
    • Create and manage VLANs for network segmentation.
    • Assign IP addresses to devices within each VLAN.
    • Test connectivity and ensure proper communication between VLANs.

    Network Topology:

    • VLAN 10: Admin/IT
      • Subnet: 192.168.0.0/26
      • Devices: PC, Printer, Access Point
    • VLAN 20: Finance/HR
      • Subnet: 192.168.0.64/26
      • Devices: PC, Printer, Access Point
    • VLAN 30: Customer Service/Reception
      • Subnet: 192.168.0.128/26
      • Devices: PC, Printer, Access Point

    Key Steps:

    • Designed a network diagram with three VLANs.
    • Configured router interfaces and set up inter-VLAN routing.
    • Created VLANs on the switch and assigned ports.
    • Assigned IP addresses to devices and configured gateways.
    • Tested connectivity and verified inter-VLAN routing.

    Tools and Technologies:

    • Cisco Packet Tracer
    • Cisco IOS

    Results:

    • Successfully set up three VLANs.
    • Improved IP address management and network security.
    • Verified connectivity and communication between VLANs.
